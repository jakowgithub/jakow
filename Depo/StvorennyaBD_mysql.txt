Створення БД mysql.

Вже встановлен mysql з користувачем root та паролем  root (які задаються під час встановлення).

В каталозі Squirrrel/lib  роблю копію БД sampledatabase, перейменовую на metro 
(в цій БД вже встановлено користувач:user1 пароль: pass1)

Подвійний клік на БД  metro, замінюю в тексті програми  classicmodels на  metro.

В командному рядку запускаю сервер БД:
C:\MySQL\MySQL Server 5.7\bin\mysqld
невеличка пауза комп'ютера далі готов до вводу

Пишу в командному рядку
mysql -u root -p < metro.sql
невеличка пауза комп'ютера далі готов до вводу
Запит пароля Password:root (відповідь root)

Захожу в Squirrrel-є БД metro.
